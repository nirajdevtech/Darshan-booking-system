Darshan Booking System

Darshan Booking System is a MERN stack web application that allows users to view temples and book darshan slots online. The system also includes organizer and admin functionalities for managing temples and bookings.

🚀 Tech Stack

Frontend

React.js

Vite

CSS

Backend

Node.js

Express.js

Database

MongoDB

📌 Features
User

View temple details

Book darshan slots

View booking history

Login and logout

Organizer

Add temples

Manage darshan slots

View bookings

Admin

Manage users

Manage organizers

Monitor platform activity

⚙️ Installation

Clone the repository

git clone https://github.com/nirajdevtech/Darshan-booking-system.git
Run Backend
cd backend
npm install
npm start

Backend runs on:

http://localhost:7000
Run Frontend
cd Frontend
npm install
npm run dev

Frontend runs on:

http://localhost:5173/uhome
